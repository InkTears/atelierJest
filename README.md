# Atelier Testing : Jest & Node.js

Cet atelier a pour but de comprendre et d'appliquer les bases des tests unitaires avec Jest dans un environnement Node.js natif. Aucune configuration complexe, l'objectif est de se concentrer sur la logique métier et la rédaction des tests.
